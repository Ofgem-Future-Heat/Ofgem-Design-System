Patterns are best-practice solutions to common design problems that users face when using Ofgem services. Use them alongside components to build pages that are consistent, accessible and easy to use.

Approve an amendment

Let case handlers review and approve amendments raised against an existing Ofgem application.

Use this pattern when a user needs to check the detail of a requested change, compare it against the current record, and either approve, reject or return it for more information.
