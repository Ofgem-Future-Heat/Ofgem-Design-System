How it works

The profile snapshot component displays application details and installer account details as a snapshot.

When to use this component

  • For internal applications and services
  • To provide snapshot of application details and installer account details
  • Extend this to include or exclude certain details as needed

When not to use this component

  • Don't use on non-internal applications and services

Research

This component follows MoJ Design System patterns and has been tested with users on LA Flex and GBIS. Headers are most effective when they provide clear navigation and consistent branding.
